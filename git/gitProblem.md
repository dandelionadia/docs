### це ЛИШЕ коли зміни не закомічені:

- **git checkout .** - прибрати всі зміни(change commit)

```
$ git reset .
$ git checkout .
```

`reset` робить зелені файли червоними

`checkout` прибирає зміни (червоні файли)
