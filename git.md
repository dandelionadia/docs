## Додати в пустий репозиторій readme

- **touch README.md**

- **git add README.md**

- **git commit -m ''**

- **git push -u origin master**

## Підключення bootstrap і scss:

* скопіювати папку;
* видалити  node_modules в корені проекту;
*  відкрий цей проект в терміналі(це згенерує папку "node_modules" заново);
* запустити gulp
